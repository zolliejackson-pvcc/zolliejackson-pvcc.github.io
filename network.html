Networking & The Internet
Network Classification
  • LAN

  • MAN

  • WAN
  1. Local Area Network (LAN) is a collection of devices connected together in one physical location, such as a building, office, or home
  2. A metropolitan area network (MAN) is a computer network that connects computers within a metropolitan area.
  3. Wide Area Network (WAN) is the technology that connects your offices, data centers, cloud applications, and cloud storage together.
Network Models
  • Bus Network

  • Star Network
  1. A bus network is a local area network (LAN) topology in which each node- a workstation or other device -- is connected to a main cable or link called a bus.
  2. A star network is a local area network (LAN) topology in which all nodes-personal computers (PCs), workstations or other devices -- are directly connected to a common central computer that is often referred to as a hub.
Protocols
  • CSMA/CD Ethernet

  • CSMA/CA Wifi
  1. Carrier-sense multiple access with collision detection (CSMA/CD) is a medium access control (MAC) method used most notably in early Ethernet technology for local area networking.
  2. Carrier sense multiple access/collision avoidance (CSMA/CA) is a protocol for carrier transmission in 802.11 networks.
Combining Network
  • Repeater

  • Bridge

  • Switch

  • Router

  • Gateway
  1. A repeater is a powerful network hardware device that regenerates an incoming signal from the sender before retransmitting it to the receiver.
  2. .A bridge in a computer network is a device used to connect multiple LANs together with a larger Local Area Network (LAN).
  3. A network switch connects devices in a network to each other, enabling them to talk by exchanging data packets.
  4. A router is a device that connects two or more packet-switched networks or subnetworks.

  5. A gateway is a network node used in telecommunications that connects two networks with different transmission protocols together.
Client/server Model
  • P2P: peer-to-peer

  • Distributed System

  • Cloud Computing
  1. In peer-to-peer (P2P) networking, a group of computers are linked together with equal permissions and responsibilities for processing data.
  2. A distributed system is simply any environment where multiple computers or devices are working on a variety of tasks and components, all spread across a network.
  3. A cloud network is a Wide Area Network (WAN) that hosts users and resources and allows the two to communicate via cloud-based technologies.
Internet Addressing
  • Tier 1

  • Tier 2

  • Tier 3
  1. A Tier 1 network is an Internet Protocol network that can reach every other network on the Internet solely via settlement-free interconnection.
  2. A Tier 2 network is an Internet service provider which engages in the practice of peering with other networks, but which also purchases IP transit to reach some portion of the Internet.
  3. A Tier 3 provider is by definition primarily engaged in delivering Internet access to end customers. Tier 3 ISPs focus on local business and consumer market conditions.
IP Address
  • ICANN

  • DNS

  • TLD

  • Country-Code TLDs
  1. ICANN is a public-private partnership that is responsible for the following functions related to internet names and numbers: Allocation of IP address space.
  2. The Domain Name System (DNS) is the phonebook of the Internet. Humans access information online through domain names like nytimes.com or espn.com. Web browsers interact through Internet Protocol (IP) addresses.
  3. TLD stands for top-level domain and describes the part of a URL that represents the final section of a domain name. For example, in the domain name Shopify.com, the TLD is .com.
  4. A country code top-level domain (ccTLD) is an Internet top-level domain generally used or reserved for a country, sovereign state, or dependent territory identified with a country code.
Internet Protocols
  • FTP

  • HTTP

  • SMTP
  1. FTP (File Transfer Protocol) is a standard network protocol used for the transfer of files from one host to another over a TCP-based network, such as the Internet.
  2. HTTP is a protocol for fetching resources such as HTML documents. It is the foundation of any data exchange on the Web and it is a client-server protocol, which means requests are initiated by the recipient, usually the Web browser.
  3. SMTP (Simple Mail Transfer Protocol) is a TCP/IP protocol used in sending and receiving email. SMTP is used most commonly by email clients, including Gmail, Outlook, Apple Mail and Yahoo Mail.
Word Wide Web
  • URL

  • Client-Side Activities

  • Server-Side Activities
  1. URL stands for Uniform Resource Locator. A URL is nothing more than the address of a given unique resource on the Web.
  2. In web development, 'client side' refers to everything in a web application that is displayed or takes place on the client (end user device).
  3. Server-side scripts run on the server instead of the client, often in order to deliver dynamic content to webpages in response to user actions.
Internet four-layer software
  • Application layer

  • Transport layer

  • Network layer

  • Link layer
  1. An application layer is an abstraction layer that specifies the shared communication protocols and interface methods used by hosts in a communications network.
  2. The transport layer is Layer 4 of the Open Systems Interconnection (OSI) communications model.
  3. The network layer uses network addresses (typically Internet Protocol addresses) to route packets to a destination node.
  4. The link layer is the lowest layer in the Internet protocol suite, the networking architecture of the Internet.
Internet is:
  • Packet-switching network

  • Fault tolerant

  • Redundant
  1. Packet switching is the transfer of small pieces of data across various networks.

  2. Fault Tolerance simply means a system's ability to continue operating uninterrupted despit the failure of one or more of its components.
  3. Network redundancy is the process of providing multiple paths for traffic so that data can keep flowing even in the event of a failure.
Cybersecurity
  • Malware

  • Virus

  • Worm

  • Trojan Horse

  • Spyware

  • Phishing

  • DDoS Attack
  1. Malware is software that is installed on a computer without the user's consent and that performs malicious actions, such as stealing passwords or money.
  2. A computer virus is a program that spreads by first infecting files or the system areas of a computer or network router's hard drive and then making copies of itself.
  3. A worm is a type of malware or malicious software that can replicate rapidly and spread across devices within a network. As it spreads, a worm consumes bandwidth, overloading infected systems and making them unreliable or unavailable.
  4. A Trojan horse is a program downloaded and installed on a computer that appears harmless, but is, in fact, malicious.
  5. Spyware is malicious software that enters a user's computer, gathers data from the device and user, and sends it to third parties without their consent.
  6. Phishing is the practice of sending fraudulent communications that appear to come from a legitimate and reputable source, usually through email and text messaging.
  7. DDos attcak is a cybercrime in which the attacker floods a server with internet traffic to prevent users from accessing connected online services and sites.
Protection
  • Firewall

  • Spam Filter

  • Proxy Server

  • AntiVirus Software

  • Cryptography

  • Public Key Encryption
  1. A firewall is a network security device that monitors incoming and outgoing network traffic and decides whether to allow or block specific traffic based on a defined set of security rules.
  2. A spam filter is a program used to detect unsolicited, unwanted and virus-infected emails and prevent those messages from getting to a user's inbox.
  3. A proxy server is a system or router that provides a gateway between users and the internet. Therefore, it helps prevent cyber attackers from entering a private network.
  4. Antivirus software (antivirus program) is a security program designed to prevent, detect, search and remove viruses and other types of malware from computers, networks and other devices. Cryptography is the process of hiding or coding information so that only the person a message was intended for can read it.
  5. In cryptography, a public key is a large numerical value that is used to encrypt data.
  6. The key can be generated by a software program, but more often, it is provided by a trusted, designated authority and made available to everyone through a publicly accessible repository or directory.
